
# MP4 to MP3 Converter

**MP4 to MP3 Converter** は、MP4動画ファイルをMP3音声フォーマットに変換するために設計された、シンプルで強力なmacOS向けGUIアプリケーションです。Python と PyQt6 で構築されており、直感的な操作性を提供します。

## ✨ 特徴

コードから読み取れる主な機能と、想定される機能は以下の通りです：

  * **GUIベースの操作:** PyQt6を使用したモダンなユーザーインターフェース。
  * **ドラッグ＆ドロップ:** ファイルをウィンドウにドラッグするだけで簡単に追加可能。
  * **バッチ変換:** 複数のファイルを一度にまとめて変換処理。
  * **進捗管理:** 変換状況を可視化。
  * **高解像度対応:** macOSのHigh DPI (Retina) ディスプレイに対応した鮮明な表示。
  * **ログ機能:** トラブルシューティングに役立つ詳細なログ出力。

## 🛠️ 必要要件

  * **Python 3.8+**
  * **FFmpeg** (変換エンジンのバックエンドとして必要になる場合が一般的です)

## 📦 インストール


```bash
pip install -r requirements.txt
```

## 🚀 使い方

アプリケーションを起動するには、以下のコマンドを実行してください。

```bash
python main.py
```

### 操作方法

1.  アプリケーションが起動します。
2.  変換したいMP4ファイルをウィンドウにドラッグ＆ドロップします（または「ファイルを開く」ボタンを使用）。
3.  変換設定を確認し、「変換開始」ボタンをクリックします。
4.  変換されたMP3ファイルは、元のファイルと同じフォルダ（または指定した出力先）に保存されます。
